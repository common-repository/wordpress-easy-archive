== Description ==

WordPress Easy Archive will create an image based archive. All the images are stored in the database to keep the speed as you used to.
== Installation ==

1. Upload the folder wordpress-easy-archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Option > Easy Archive to configure plugin output.
1. Add the mark <!--archive--> to your page or post in the html interface.


== Upgrade ==

1. Delete the old plugin folder `wordpress-easy-archive`.
1. Upload the folder wordpress-easy-archive to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Option > Easy Archive to configure plugin output.
